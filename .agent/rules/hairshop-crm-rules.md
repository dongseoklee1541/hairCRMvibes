# 미용실 고객관리 서비스 개발 규칙

이 파일은 별도 규칙을 중복해서 관리하지 않습니다.

- 최신 개발 규칙의 단일 기준은 저장소 루트의 `AGENTS.md`입니다.
- 작업을 시작하기 전에 루트 `AGENTS.md`를 먼저 읽고 Plan → Approval → Implementation → Verification → Report 순서를 따릅니다.
- UI 변경은 `pencil-hairshopcrm.pen`을 먼저 업데이트하고 최소 44×44px 터치 영역과 모바일 검증 기준을 지킵니다.
- 검증 도구는 루트 `AGENTS.md`와 현재 `package.json`에 실제로 존재하는 명령만 사용합니다.
