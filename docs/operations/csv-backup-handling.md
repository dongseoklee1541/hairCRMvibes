# CSV 백업 파일 취급 절차

## 목적과 적용 범위

`/settings`의 고객 CSV와 예약 CSV는 데이터 이관·운영 점검을 위한 민감정보 내보내기입니다. 연락처, 메모, 예약 이력과 감사 필드가 포함될 수 있으므로 일반 문서처럼 보관하거나 공유하지 않습니다.

이 CSV는 Auth 사용자, 데이터베이스 객체, RLS, 파일 저장소, 트랜잭션 시점 복구를 포함하는 완전한 재해복구 백업이 아닙니다. 데이터베이스 복구 수단을 대체하지 않습니다.

## 내보내기 전 확인

1. 원장 계정으로 로그인했는지 확인합니다.
2. 저장할 기기의 디스크 암호화가 활성화되어 있는지 확인합니다.
   - macOS: FileVault
   - Windows: BitLocker 또는 조직이 승인한 동등한 암호화
3. 공용·공유 기기, 개인 메신저, 이메일 첨부, 승인되지 않은 USB와 개인 클라우드에는 저장하지 않습니다.
4. 필요한 데이터셋과 사용 목적을 먼저 정하고, 필요하지 않은 파일은 만들지 않습니다.
5. CSV는 여러 페이지를 순서대로 조회하는 운영 내보내기이며 데이터베이스 시점복구 스냅샷이 아닙니다. 대량 내보내기 중에는 고객·예약 편집을 잠시 멈추고, 완료 후 정상 종료 안내를 확인합니다.

## 다운로드 직후

1. 파일을 다운로드 폴더에 계속 두지 말고 즉시 기기 암호화 저장소나 조직이 승인한 암호화 보관 위치로 옮깁니다.
2. 파일 접근 권한을 업무상 필요한 최소 인원으로 제한합니다.
3. 파일명에 고객 이름, 전화번호 등 개인정보를 추가하지 않습니다.
4. 지원하는 데스크톱 브라우저에서는 서버 응답을 선택한 파일로 직접 스트리밍합니다. 그 외 브라우저는 호환성을 위해 메모리 Blob 다운로드를 사용하므로, 큰 파일을 내보낼 때 브라우저 메모리 사용량을 확인합니다.

## 보관과 삭제

- 최대 보관 기간은 생성일로부터 30일입니다. 업무가 끝났다면 30일을 기다리지 않고 즉시 삭제합니다.
- 삭제할 때 원본 보관 위치뿐 아니라 다운로드 폴더, 휴지통, 동기화된 클라우드 사본과 버전 기록도 함께 확인합니다.
- 운영상 30일보다 긴 보관이 필요하면 CSV를 그대로 연장 보관하지 말고 목적, 책임자, 접근자, 암호화 방식, 새 만료일을 별도로 승인받습니다.
- 애플리케이션은 다운로드 이후의 기기 파일을 추적하거나 강제 삭제할 수 없습니다. 최종 삭제 확인은 파일을 생성한 운영자의 책임입니다.

## 사고 대응

잘못된 위치에 저장했거나 승인되지 않은 사람에게 노출됐을 가능성이 있으면 다음 순서로 처리합니다.

1. 해당 파일과 동기화 사본의 추가 공유를 중지합니다.
2. 가능한 모든 사본을 삭제하되, 사고 조사에 필요한 증거 보존 지시가 있으면 그 지시를 우선합니다.
3. 노출된 데이터셋, 생성 시각, 저장·전송 경로, 접근 가능자를 기록해 운영 책임자에게 즉시 알립니다.
4. 필요하면 관련 계정 세션 종료, 권한 점검, 고객 통지와 법적 대응 범위를 별도로 판단합니다.

## 기술적 보호 경계

- `/api/export`는 사용자 JWT와 기존 RLS를 사용하며 service-role key를 사용하지 않습니다.
- 응답은 `private, no-store`이고 서비스워커·Cache Storage에 저장하지 않습니다.
- 서버는 CSV 파일을 디스크나 외부 저장소에 보관하지 않습니다.
- UTF-8 BOM, CRLF와 spreadsheet formula injection 방어는 파일 호환성과 열기 단계의 위험을 줄이지만, 파일 자체를 암호화하지는 않습니다.
- 페이지 정렬에는 변경되지 않는 `created_at`, `id`를 사용해 일반적인 예약 수정이 페이지 경계를 흔들지 않게 하지만, 여러 요청에 걸친 완전한 시점 일관성까지 보장하지는 않습니다.

## 운영 확인표

- [ ] 원장 계정과 내보낼 데이터셋을 확인했습니다.
- [ ] 암호화된 승인 기기에 저장했습니다.
- [ ] 다운로드 폴더에 원본이 남지 않았습니다.
- [ ] 개인 메신저·이메일·비승인 클라우드로 공유하지 않았습니다.
- [ ] 사용 종료일 또는 최대 30일 이내 삭제일을 기록했습니다.
- [ ] 삭제 시 휴지통·클라우드 사본·버전 기록까지 확인했습니다.
